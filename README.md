# Simple Interest Calculator

A simple bash script project to calculate simple interest.

## Description

This project provides a calculator that computes simple interest using principal amount, rate of interest, and time period entered by the user.

The formula used is:

Simple Interest = (Principal × Rate × Time) / 100

## Features

- Calculate simple interest
- User-friendly bash script
- Accepts user input from terminal
- Lightweight and easy to run

## Requirements

- Linux or Unix terminal
- Bash shell

## Usage

Clone the repository:

```bash
git clone https://github.com/Hassan-ali-mohammed/github-final-project.git
```

Go to the project directory:

```bash
cd github-final-project
```

Run the script:

```bash
bash simple-interest.sh
```

## Example

Input:

```text
Principal amount: 1000
Rate of interest: 5
Time period: 2
```

Output:

```text
Simple Interest = 100
```
