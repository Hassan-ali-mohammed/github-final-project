# Contributing to Simple Internet Calculator

Welcome! We are glad you are here. All contributions, bug reports, bug fixes, documentation improvements, enhancements, and ideas are welcome.

## How to Contribute
1. Fork the repository.
2. Create a new branch for your changes (`git checkout -b feature-branch`).
3. Commit your updates (`git commit -m "Add some feature"`).
4. Push to your branch (`git push origin feature-branch`).
5. Open a Pull Request for review.
